## Module 04 – Basic Prompt Engineering and Conversation Templates

In this module you will learn:
- What prompts are and why they matter.
- How to write simple, clear prompts for AI.
- How to start a conversation in a structured way using templates.

### Related files (Hebrew content in `../hebrew/`)

- `../hebrew/README.md` – general module description (Hebrew).  
- `../hebrew/01_intro_to_prompts.md` – introduction to prompt engineering in simple language. (~10–15 min, beginner)  
- `../hebrew/02_conversation_templates.md` – ready-to-use conversation templates (problem solving, text improvement, brainstorming, etc.). (~15–20 min, beginner)

Focus especially on `../hebrew/02_conversation_templates.md`: pick one or two templates and reuse them in your own AI chats.
